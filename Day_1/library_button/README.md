 Button Component Library

Create 6 different button variations:
- Primary (green background)
- Secondary (white background, green border)
- Danger (red theme)
- Success (green theme)
- Large size
- Small size

Requirements:
- Consistent padding
- Proper font weight
- Rounded corners
- Different colors
